## 🌟 專案簡介 | Project Overview | プロジェクト概要

**[ZH]**  
這是一個專為 AKB48 劇場觀劇與應援記錄而設計的非官方粉絲小工具。  
使用者可以在劇場座位圖上選擇座位或站位，加入公演名稱、活動日期、顯示名稱、號碼與入場輪次，並輸出成清晰的座位記錄圖片。  
它的目的不是取代官方資訊，而是讓每一次入場、每一個位置、每一段劇場回憶，都可以用更漂亮、更容易分享的方式保存下來。

**[EN]**  
AKB48 Theater Seat Memo is an unofficial fan-made web tool designed for recording theater seats and standing positions.  
Users can select a seat or standing slot on the theater map, add performance details, event date, display name, number, and entry round, then export the result as a clean seat memo image.  
It is not intended to replace official information; it is built to help fans preserve and share their theater memories in a simple, polished way.

**[JP]**  
AKB48 Theater Seat Memo は、劇場での座席や立ち見位置を記録するために作られた非公式ファンツールです。  
劇場座席表から位置を選択し、公演名、日時、表示名、番号、入場順などを入力して、見やすい座席メモ画像として保存できます。  
公式情報を代替するものではなく、劇場での一回一回の思い出を、より美しく、共有しやすい形で残すためのツールです。

## ✨ 核心亮點 | Technical Highlights | 技術的特徴

### 1. 互動式劇場座位圖 | Interactive Theater Map | インタラクティブ座席表

* **[ZH]** 使用者可以直接點選座位或站位，系統會即時標示目前選擇的位置。
* **[EN]** Users can tap or click a seat or standing slot, and the selected position is highlighted instantly.
* **[JP]** 座席または立ち見位置をクリック・タップすると、選択位置が即時に表示されます。

### 2. 應援感粉紅箭咀 | Concert-style Pink Marker | 応援風ピンク矢印

* **[ZH]** 以粉紅箭咀清楚指出「我的位置」，保留劇場應援與分享用圖的視覺感。
* **[EN]** A pink arrow marker clearly points to "my position", giving the exported image a fan-support style.
* **[JP]** ピンクの矢印で「自分の位置」を分かりやすく示し、応援画像らしい雰囲気を演出します。

### 3. 手機友善操作 | Mobile-friendly Experience | スマートフォン最適化

* **[ZH]** 手機版已壓縮標題區高度、優化座位圖工具列及表單密度，讓第一屏更快看到座位圖與主要操作。
* **[EN]** The mobile layout uses a compact header, tighter map toolbar, and denser form spacing so the seat map appears earlier on screen.
* **[JP]** モバイル版ではヘッダー、座席表ツールバー、入力フォームの余白を調整し、座席表と主要操作がすぐ見えるように最適化しています。

### 4. 圖片輸出與分享 | Image Export & Share Link | 画像保存・共有リンク

* **[ZH]** 可輸出座位記錄圖片，亦可複製分享連結，方便日後保留或傳送給朋友。
* **[EN]** Export the current memo as an image or copy a share link to preserve and send the selected state.
* **[JP]** 現在の座席メモを画像として保存したり、共有リンクとしてコピーしたりできます。

## 📝 功能說明 | Features | 機能

### 基本資料輸入 | Basic Information | 基本情報入力

* **[ZH]** 可填寫公演名稱、活動日期及時間、顯示名稱、號碼、入場輪次。
* **[EN]** Supports performance name, event date and time, display name, number, and entry round.
* **[JP]** 公演名、日時、表示名、番号、入場順を入力できます。

### 座位選擇 | Seat Selection | 座席選択

* **[ZH]** 支援座位與站位選擇，並可清除目前選擇重新開始。
* **[EN]** Supports seat and standing-position selection, with a clear option to reset the current choice.
* **[JP]** 座席と立ち見位置の選択に対応し、選択内容をクリアしてやり直すこともできます。

### 縮放控制 | Zoom Control | ズーム操作

* **[ZH]** 座位圖可放大或縮小，方便在不同螢幕尺寸下查看細節。
* **[EN]** The seat map can be zoomed in or out for easier viewing across different screen sizes.
* **[JP]** 座席表を拡大・縮小でき、画面サイズに合わせて細部を確認しやすくなっています。

### Safari / iPhone 儲存支援 | Safari / iPhone Save Support | Safari / iPhone 保存対応

* **[ZH]** 在 iPhone / Safari 上會顯示圖片預覽，使用者可長按圖片保存。
* **[EN]** On iPhone / Safari, an image preview opens so users can long-press to save it.
* **[JP]** iPhone / Safari では画像プレビューを表示し、長押しで保存できます。
