# AKB48 Seat Memo v3.0.0 SVG Master

This version uses the uploaded Illustrator-exported `seatmap.svg` as the canonical map artwork.

## Key points
- ViewBox: 760 × 640
- Base map: `assets/seatmap.svg`
- Clickable hit areas are generated from the Illustrator selectable layers.
- Non-selectable stage marks remain visual only.
- Export draws the SVG map directly to a high-resolution PNG, then adds the pink arrow and footer.
